| # | Model/Method | Source | URL | License | Format | Notes |

**TOP RECOMMENDATION:** The **Mixamo + Combiner workflow** is the gold standard:
1. Download MadTrollStudio or OpenGameArt soldier model
2. Upload to Mixamo.com
3. Apply walk, run, rifle_idle, rifle_shoot, death animations
4. Export each as FBX
5. Combine via mixamo2gltf.com into single GLB
6. Load in Three.js with `AnimationMixer`
